# Skills

Favorite Claude Code skills.

## What's here

| Folder | Description |
|--------|-------------|
| `learn/` | Constraint-driven learning skill — explains WHY things had to be the way they are, then produces a reveal.js slide deck and NotebookLM podcast |
| `earnings-preview/` | Pre-earnings analysis: beat/miss history, guidance patterns, consensus estimates, and a mini projection model |
| `earnings-review/` | Post-earnings analysis: beat/miss outcomes, new guidance, performance trajectory, and analyst commentary |
| `hedge-fund-analyst/` | Domain expertise for hedge fund equity analysis — investment philosophy and judgment framework for stocks, earnings, and valuations |
